The technology behind 3d printer is quite awesome. We are here to make it simple.

3D printing technology is know as a manufacturing method under Mechanical Engineering.
It is also known as additive Manufacturing.

Usually, it is a process of building a three dimensional object from computer aided to a step by step process leads to the product manufacture.

The steps are ordered as shown below

  •  3D model Designing
          A  Three dimensional object is designed using softwares such as solid works, catia,                    Autofusion 360 etc.,.
  • STL file creation
          STL (a shortened form of "stereolithography") is a document position local to the                        stereolithography CAD programming made by 3D Systems. STL has a few backronyms,             for example, "Standard Triangle Language" and "Standard Tessellation Language". This             record organization is upheld by numerous other programming bundles; it is generally             utilized for fast prototyping, 3D printing and Computer Aided Production.  CAD model               is converted to a STL file to tessellate the 3D shape and slice it into digital layers.
  •  STL file transfer
           STL file is then transferred to the printer using custom machine software.The STL file                 transfer is the process which makes the printer to understand the technology that how             many prints are to made and other technical information.
  • Machine set up
           Consumables that is the printing material which is to be preloaded on the printer.This               is the process behind machine set up and the required connection and backups                         process are made.
  •  Build
           Printer constructs or prints the model by applying layer by layer the consumables and               it make the product.
  • Part Removal
           The product is removed from the printing machine after the product is finished                           printing.
  • Post processing
           Finally, post processing, such as cleaning, removing unwanted materials , quality                       check,and painting.
