Application Mainspring

Module : Metrics Actual

    Requirements:

  1. Navigate to CFO_Onsite Project.
  2. Select Metrics Actual in Execute phase.
  3. Navigate to Measures tab in Metrics Actual.
  4. Review the functionalities of different Measures tables.

    Detailed Description:

  1. Select CFO_Onsite project in Projects/Programs :

    User should be able to select CFO_Onsite project by logging into Mainspring Application using valid credentials.

  2. Select Metrics Actual in Execute phase :

    User should be able to navigate to Metrics Actual from Execute Dropdown menu.

  3. Navigate to Measures tab in Metrics Actual :

    User should be able to select Measures tab in Metrics Actual module.

  4. Review the functionalities auch as Maximize/Minimize, Collapse/Expand, Filter options tables :

    User should be able to check the functionalities of different tables.
